# Reflection

I built an AI-Powered Feature using Groq Cloud. I had already included authentication on my midterm project using Supabase to store and authenticate data. I wanted to try adding AI to the webpage.

The thing that surprised me the most was how easy it was to set up the AI feature. All you need is an API key — but at the same time it could be dangerous, because with the API key the AI is gaining access to everything.

Learning about and implementing this feature was eye-opening because there were a lot of errors I ran into. Mostly being a credit or payment plan issue. I occasionally ran into an instance of an LLM model being out of date as well.

I would try to improve the AI to handle bigger tasks — maybe create a chatbot.
